# OpenSeasame
It features:
* a low-power STM32L072CBT MCU
* a 128x64 OLED SSD1306 display
* an RGB LED for notifications
* a piezoelectric speaker
* a 1000mAh battery
* a USB C connector for data and charging
* three buttons
